使用別人寫好的插件

插件 ： VueAxios

其實在使用別人寫好的插件時，都會附使用說明，就照著說明使用即可。

就目前而言，比較常用的 Vue 插件就是 Vue-router 跟 Vuex ，但因為這2個插件只要檢測到存在 Vue 這個變量後，就會自動調用 Vue.use() ，所以不需要額外寫 Vue.use() ，但必須注意如果是使用 CommonJS 模塊化，仍得使用 Vue.use() 。
