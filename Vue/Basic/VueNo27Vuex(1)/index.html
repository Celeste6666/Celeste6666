了解了如何創建基礎的 Vue Router 後，就應該來學學如何利用 Vuex 來進行全域狀態管理。

為什麼要使用 Vuex

Vuex 官網是這麼介紹：

Vuex 是一个专为 Vue.js 应用程序开发的状态管理模式。它采用集中式存储管理应用的所有组件的状态，并以相应的规则保证状态以一种可预测的方式发生变化。

Vuex 的概念就是建立一個集中管理的資料庫，不論哪個模組需要連動的資料都可以向 Vuex 取得。
